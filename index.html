Maximum Depth: the maximum depth that the program will draw squares within squares. For example a maximum depth of 5 means that it is possible to have a square within a square within a square... up to five times.

Chance of Splitting: the chance out of 100 that any given square will be split into four. For example, setting the chance to 0 will just draw one big square, and setting the chance to 100 will split wherever possible

Width of Split Line: the width of the black lines which divide up the image. I would recommend using a smaller width but you are free to experiment with it anyway you like.
